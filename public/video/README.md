# Hero Video

## Format

- Location: `public/video/hero-arca-mp4/Epistolario.mp4`
- Public URL: `/video/hero-arca-mp4/Epistolario.mp4`
- Used by: Homepage hero (Variant B, accessed via `/?hero=video`)

## Requirements

- Format: MP4 (H.264)
- Resolution: 1920x1080 minimum recommended
- Duration: 15-30 seconds ideal (loops continuously)
- Audio: None (played muted)
- File size: Keep under 15MB for fast loading
- Content: Atmospheric footage relevant to the archive/epistolary theme

## Fallback behavior

If the video fails to load within 3 seconds, the homepage silently falls back to the editorial variant (Variant A) with the warm parchment background.
